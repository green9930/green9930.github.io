label의 명시적 연결, 암묵적 연결

Posted on 2020-10-26 In HTML/CSS

What is Label?

label 요소란 입력 양식(form) 내부의 입력 요소(input)에 이름을 지정해주는 태그.
label과 input 사이를 연결해주는 방식은 크게 명시적 연결과 암시적 연결로 나뉜다.

  • 웹접근성 측면에서의 label
    화면리더기(screenreader)가 form 내부에서 label을 읽어 보조기술(assistive technology) 사용자가 입력해야 하는 텍스트가 무엇인지 잘 파악할 수 있게 도와준다.

  • hit area의 이점
    관련 label을 클릭해서 input을 활성화시킬 수 있으며 보다 넓어진 활성화 가능 면적은 사용자들의 편의성을 증가시킨다.
    (EX. radio나 checkbox처럼 입력 면적이 적은 곳에서 유용하게 쓸 수 있다.)

  • label 요소의 주의사항
    label 내부에 <a>, <button>와 같은 인터랙티브 컨텐츠 배치를 피한다. 이 경우 label 요소가 들어간 form 형식을 활성화하기 어렵기 때문.



label 태그의 명시적(explicit) 연결

<label> 태그의 for 속성과 <input> 태그의 id 속성을 동일하게 입력하여 명시적으로 연결시켜주는 방법
명시적 연결은 label과 input 사이에 다른 html 요소가 포함될 경우 많이 사용된다.
디자인상 편의성을 위해 명시적 연결을 권장한다.



label 태그의 암시적(implicit) 연결

<input> 태그를 <label> 태그로 감싸서 for, id 속성은 필요없이 암시적으로 연결시켜주는 방법
